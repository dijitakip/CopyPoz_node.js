## 🎯 Container Tanımlama Yöntemleri

### 1. Container Adı ile Bulma
```bash
# CopyPoz container'larını adıyla bul
docker ps --filter "name=copypoz"

# Sonuç:
# NAMES              IMAGE              STATUS
# copypoz_app        copypoz-app        Up 2 minutes
# copypoz_mysql      mysql:8.0          Up 3 minutes
```

### 2. Label ile Bulma
```bash
# Docker Compose tarafından oluşturulan label'ları kullan
docker ps --filter "label=com.docker.compose.project=copypoz"

# Sonuç:
# NAMES              IMAGE              STATUS
# copypoz_app        copypoz-app        Up 2 minutes
# copypoz_mysql      mysql:8.0          Up 3 minutes
```

### 3. Network ile Bulma
```bash
# copypoz_network'e bağlı container'ları bul
docker network inspect copypoz_copypoz_network

# Sonuç:
# "Containers": {
#   "copypoz_app": {...},
#   "copypoz_mysql": {...}
# }
```

### 4. Volume ile Bulma
```bash
# copypoz_mysql_data volume'ünü kullanan container'ları bul
docker ps -a --filter "volume=copypoz_copypoz_mysql_data"

# Sonuç:
# NAMES              IMAGE              STATUS
# copypoz_mysql      mysql:8.0          Up 3 minutes
```

## 🔄 Lifecycle

### Başlatma Sırası
```
1. docker-compose up -d
2. MySQL container başlar
3. MySQL healthcheck çalışır
4. App container başlar (MySQL healthy olduktan sonra)
5. App bağlantı kurar
6. Sistem ready
```

### Durdurma Sırası
```
1. docker-compose down
2. App container durur
3. MySQL container durur
4. Network temizlenir
5. Sistem kapalı
```

## 🎯 Otomatik Bulma Scripti

### PowerShell Script
```powershell
# CopyPoz container'larını bul ve göster
function Find-CopyPozContainers {
    Write-Host "CopyPoz Container'ları Aranıyor..." -ForegroundColor Green
    
    $containers = docker ps -a --filter "name=copypoz" --format "{{.Names}}"
    
    if ($containers) {
        Write-Host "Bulundu:" -ForegroundColor Green
        $containers | ForEach-Object {
            $status = docker ps --filter "name=$_" --format "{{.Status}}"
            Write-Host "  ✓ $_" -ForegroundColor Green
            Write-Host "    Status: $status" -ForegroundColor Cyan
        }
    } else {
        Write-Host "CopyPoz container'ları bulunamadı" -ForegroundColor Red
    }
}

Find-CopyPozContainers
```

### Bash Script
```bash
#!/bin/bash

echo "CopyPoz Container'ları Aranıyor..."

containers=$(docker ps -a --filter "name=copypoz" --format "{{.Names}}")

if [ -z "$containers" ]; then
    echo "CopyPoz container'ları bulunamadı"
    exit 1
fi

echo "Bulundu:"
echo "$containers" | while read container; do
    status=$(docker ps --filter "name=$container" --format "{{.Status}}")
    echo "  ✓ $container"
    echo "    Status: $status"
done
```
